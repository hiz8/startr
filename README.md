Startr
======

Set up local development environment base on Vagrant, Ansible, Node.js.

## Quick start

You can run the playbooks with Vagrant.

```bash
$ vagrant up
```

## Development environment

- CentOS (6.8 x86_64)
- Ansible (latest)
- Git (latest)
- nvm (0.26.1)
- Node.js (4.1)
