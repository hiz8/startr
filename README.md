# Startr

Set up local development environment base on Vagrant, Ansible, Node.js.

## Versioning

-   CentOS 7 (master branch)
-   CentOS 6 (CentOS_6 branch)

## Quick start

You can run the playbooks with Vagrant.

```bash
$ vagrant up
```

## Development environment

-   CentOS (7 x86_64)
-   Ansible (latest)
-   Git (latest)
-   nvm
-   Node.js
